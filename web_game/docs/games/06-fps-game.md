# 🔫 FPS 射击 (战术突击) — 游戏架构文档

## 概述
第一人称射击游戏，支持角色选择、地图选择、多游戏模式、6种武器、跳跃/滑铲、AI对手。平台中技术最复杂的游戏。

## 技术栈
- 3D引擎: Three.js v0.160.0 (ES Module, unpkg CDN)
- 后处理: EffectComposer + UnrealBloomPass + OutputPass
- 视角控制: Pointer Lock API
- 物理: 自实现重力/碰撞 (无物理引擎)
- 音效: Web Audio API (OscillatorNode + BufferSource白噪声)

## 关键技术点

### 1. 程序化纹理生成
使用Canvas生成各种PBR纹理，作为Three.js材质的map贴图:

| 纹理 | 生成方法 |
|------|---------|
| 混凝土 | 灰色底 + 6000个随机噪点 |
| 地砖 | 64px格子 + 随机亮度 + 网格描边 |
| 沙地 | 米色底 + 5000个暖色噪点 |
| 砖墙 | 32×16砖块 + 行偏移 + 随机色差 |
| 金属 | 灰色底 + 200条水平纹路 |
| 木箱 | 棕色底 + 对角十字 + 噪点 |

### 2. 后处理管线
```
RenderPass (渲染场景) → UnrealBloomPass (泛光, 强度0.25) → OutputPass (输出校正)
```
Bloom 让发光物体 (枪口火焰、靶标、灯光) 产生柔和光晕。

### 3. 滑铲机制
```javascript
if (冲刺中 && 按C && 地面上 && 速度>3) {
  sliding = true;
  slideTimer = 0.6秒;
  crouching = true;  // 视角降低
  // 保持当前速度惯性滑行
}
// slideTimer倒计时结束后恢复
```

### 4. 射线检测命中
```javascript
for (每条射线) {
  direction = 视线方向 + 随机散布偏移;
  raycaster.set(相机位置, direction);
  hits = raycaster.intersectObjects(所有目标, true);
  // 向上遍历parent找到根Group获取userData
  // 爆头: hit.point.y > target.position.y + 1.3
}
```
