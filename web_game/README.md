# MyWeb 游戏平台 — 完整使用文档

## 一、项目简介

MyWeb 游戏平台是一个基于 Node.js 的 Web 游戏合集，包含 **7 款** 不同类型的浏览器游戏。全部游戏采用原生 HTML/CSS/JavaScript 开发，5 款使用 Canvas 2D 渲染，2 款使用 Three.js 3D 引擎。无需安装任何游戏客户端，打开浏览器即可畅玩。

---

## 二、环境要求与安装

### 2.1 环境要求

| 项目 | 最低要求 | 推荐 |
|------|----------|------|
| Node.js | v18+ | v20+ / v24+ |
| npm | v8+ | v10+ |
| 浏览器 | Chrome 90+ / Firefox 90+ / Edge 90+ | 最新版 |
| 网络 | FPS 和探索游戏首次需联网（加载 Three.js CDN） | 其他 5 款无需联网 |

### 2.2 安装步骤

```bash
# 进入项目目录
cd web_game

# 安装依赖（仅首次需要）
npm install

# 启动服务器
npm start
```

看到 `Server running at http://localhost:3000` 后，在浏览器打开该地址。

### 2.3 启动方式

| 命令 | 说明 |
|------|------|
| `npm start` | 生产模式启动 |
| `npm run dev` | 开发模式（修改文件自动重启） |
| `PORT=8080 npm start` | 自定义端口 |

### 3.3 服务器工作方式

Express.js 服务器负责：
1. **主页渲染**：EJS 模板编译为 HTML
2. **静态文件托管**：将 `games/` 目录下的 HTML 文件直接分发给浏览器
3. **API 接口**：预留 JSON API（`/api/hello`）

每款游戏是**完全独立的单文件 HTML 应用**，可以脱离服务器直接在浏览器中打开运行。

## 四、游戏列表与操作指南

### 4.1 🦆 抓大鹅（三消堆叠）

**玩法**：三层堆叠方块，点击可见的方块移入底部槽位，三个相同图案自动消除。

| 操作 | 说明 |
|------|------|
| 点击方块 | 将可见（未被遮挡 60% 以上）的方块移入槽位 |
| 移出按钮 | 移除槽位最后一个方块（每关 1 次） |
| 撤回按钮 | 撤销上一步操作（每关 1 次） |
| 洗牌按钮 | 重新随机分配剩余方块的图案（每关 1 次） |

**特色**：
- 底部 7 个占位框清晰显示剩余容量
- 明亮的卡片配色（可点击：白色卡片，不可点击：暗灰色）
- 只要方块露出超过 40% 就可以点击
- 消除时全屏金色粒子爆炸特效
- 3 个难度递增关卡（6/8/10 种图案）

**胜负条件**：
- ✅ 清除所有方块 → 过关
- ❌ 槽位满 7 个且无法消除 → 失败

### 4.2 ♟️ 中国象棋

**玩法**：标准中国象棋双人对战，完整走法规则。

| 操作 | 说明 |
|------|------|
| 点击棋子 | 选中（显示绿色可走位置） |
| 点击目标位置 | 移动棋子 |
| 悔棋按钮 | 撤销上一步 |
| 新局按钮 | 重新开始 |

**特色**：
- 木纹渐变棋盘 + 3D 立体棋子（径向渐变 + 高光）
- 完整规则：蹩马腿、塞象眼、炮跳吃、将帅对面
- 将军检测 + 将死/困毙判定
- 走棋记录面板（可滚动查看历史）
- 上一步高亮显示

### 4.3 🔍 捉迷藏（找物游戏）

**玩法**：在场景中找到所有隐藏的 emoji 物品。

| 操作 | 说明 |
|------|------|
| 点击场景 | 在物品位置点击即找到 |
| 右侧面板 | 显示需要寻找的物品清单 |

**特色**：
- 3 个程序化绘制场景：厨房 / 花园 / 书房
- 发现物品：绿色发光环 + 双音上升音效
- 点错位置：红色 ✕ 标记
- 计分：物品分 + 时间奖励 + 准确奖励

---

### 4.4 🧐 找不同

**玩法**：左右两张图，找出 5 处差异。

| 操作 | 说明 |
|------|------|
| 点击差异位置 | 在任意一张图上点击即可 |
| 提示按钮 | 闪烁高亮一处差异（每关 3 次，扣 30 分） |

**特色**：
- 3 个关卡：城市街景 / 海底世界 / 太空站
- 差异标记同步显示在两张图上
- 计时系统 + 时间奖励

---

### 4.5 🍬 开心消消乐

**玩法**：8×8 网格，交换相邻方块消除 3 个以上相同图案。

| 操作 | 说明 |
|------|------|
| 点击第一个方块 | 选中（金色边框） |
| 点击相邻方块 | 交换；如果产生匹配则消除 |

**特色**：
- **无限关卡**：关卡数无上限，自动递增难度
  - 图案种类：6 种 → 最多 10 种（🍎🍊🍋🍇🍓🫐🍒🥝🍑🥭）
  - 目标分数：逐关递增（400 + 关卡 × 250）
  - 可用步数：逐关递减（32 → 最少 18）
- **大消除奖励**：
  - 4 连消：额外奖励分数 + "✨ 4连消!" 青色提示
  - 5+ 连消：大额奖励 + "💥 5连消!" 红色提示 + **额外奖励 1 步**
  - 连锁消除：分数倍率递增（×2、×3...）
- 重力掉落填补空缺
- 无有效走法时自动洗牌
- 明亮的格子背景（棋盘格纹理 + 阴影）

### 4.6 🔫 FPS 射击（战术突击）

**玩法**：第一人称射击，限时消灭靶标和敌人。

**开局选择**：
- **4 个角色**：突击兵🔥 / 侦察兵👁 / 重装兵🛡 / 医疗兵💊
- **3 张地图**：仓库🏭 / 沙漠🏜 / 工厂⚙
- **3 种模式**：自由射击🎯 / 团队竞技⚔ / 拆弹模式💣

| 按键 | 动作 |
|------|------|
| WASD | 移动 |
| 鼠标移动 | 控制视角 |
| 左键 | 射击 |
| Space | 跳跃 |
| C | 蹲下（冲刺中按 C = 滑铲） |
| Shift | 冲刺 |
| R | 换弹 |
| 1-6 | 切换武器 |
| E | 拆弹（拆弹模式，靠近炸弹时） |
| ESC | 解锁鼠标 |

**6 种武器**：

| # | 名称 | 弹夹 | 特性 |
|---|------|------|------|
| 1 | Pistol | 15 | 半自动，无限备弹 |
| 2 | Rifle | 30 | 全自动，中距离 |
| 3 | Shotgun | 8 | 7 射线散射 |
| 4 | SMG | 35 | 超快射速 |
| 5 | Sniper | 5 | 高伤害 80，慢射速 |
| 6 | LMG | 60 | 大弹夹压制 |

**特色**：
- Bloom 后处理 + ACES 色调映射
- 程序化 PBR 纹理（混凝土 / 地砖 / 沙地 / 砖墙 / 金属 / 木箱）
- 6 种不同外形的第一人称武器视模
- AI 敌人（追踪 + 射击 + 巡逻 + 死亡重生）
- 团队模式下 3 个 AI 队友自主作战
- 拆弹模式：按 E 拆弹（3 秒），成功 +500 分
- 圆形小地图雷达
- 击杀信息流 + 命中标记 + 爆头判定

## 六、扩展指南

### 新增游戏

1. 在 `games/` 下创建新目录，放入 `index.html`
2. 在 `views/index.ejs` 中添加新的游戏卡片
3. 重启服务器即可

### 替换素材

将图片放入 `assets/images/`，在游戏代码中使用 `new Image()` 加载替代当前的 emoji / Canvas 绘制。

### 部署

项目可直接部署到任何支持 Node.js 的平台：
- **本地**: `npm start`
- **Vercel**: 直接连接 Git 仓库
- **Railway / Render**: 自动检测 Node.js 项目
- **自有服务器**: `PORT=80 npm start`

## 七、技术亮点

| 亮点 | 说明 |
|------|------|
| **零外部素材** | 全部图形用 Canvas/Three.js 代码绘制，emoji 做图标，无需加载图片 |
| **程序化音效** | Web Audio API 振荡器 + 白噪声合成，零音频文件 |
| **程序化纹理** | Canvas 生成 PBR 材质贴图（混凝土、砖墙、草地、金属等） |
| **单文件游戏** | 每款游戏一个 HTML 文件包含全部代码，可独立运行 |
| **无限关卡** | 消消乐关卡由算法动态生成，难度无上限递增 |
| **后处理管线** | FPS 游戏使用 Bloom 泛光 + ACES 色调映射 |

---

## 八、已知限制

1. **FPS 和探索游戏不支持移动端**（依赖 Pointer Lock API + 键盘）
2. **高分数据仅在本地浏览器**（无服务器存储）
3. **Three.js 游戏需联网**（首次从 unpkg CDN 加载 ~600KB，之后浏览器缓存）
4. **场景为程序化绘制**（非照片级画面）

## 九、附属项目

### Unity FPS 项目

位于 `unity-fps-project/` 目录，包含 10 个 C# 脚本：

- `PlayerController.cs` — 移动 + 跳跃 + 冲刺 + 滑铲
- `PlayerCamera.cs` — 鼠标视角 + 后坐力
- `PlayerHealth.cs` — HP + 护甲
- `WeaponBase.cs` — 武器基类（射速 / 弹药 / 后坐力 / 射线检测）
- `WeaponManager.cs` — 武器切换
- `AIController.cs` — AI 状态机（巡逻 / 追击 / 攻击 / 掩护）
- `GameModeBase.cs` — 游戏模式基类
- `TeamDeathmatch.cs` — 团队竞技
- `BombDefusal.cs` — 拆弹模式
- `HUDManager.cs` — 完整 HUD 系统

使用 Unity 2022.3+ 打开 `unity-fps-project` 目录，详见其 `README.md`。
